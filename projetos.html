Meus Projetos

Projeto 01 | Jogo da Forca

Este é um projeto feito 100% em Java 8. O intuito deste jogo é proporcionar a exata experiência do Jogo da Forca convencional. Este projeto também conta com persistência de dados feita através de um arquivo XML, logo, usuários comuns e o administrador cadastrados, as palavras cadastradas não são perdidos após a execução do programa for encerrada. Todas as interfaces gráficas foram desenvolvidas com o uso da API javax.swing. Clique aqui para acessar o repositório para este projeto.

Projeto 02 | Algoritmos de Ordenação, Busca e manipulação de matrizes.

Este é um repositório de scripts escritos em python para a manipulação da estrutura de dados matriz, algorítmos de ordenação - sendo eles: Selection Sort, Insertion Sort, Bubble Sort e Merge Sort - e algorítmos de busca binária e linear. Clique aqui para acessar este repositório.
